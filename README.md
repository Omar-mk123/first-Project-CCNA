# Basic Network Subnetting & Routing

## 📌 Description
This project demonstrates a simple network divided into two subnets using a router.

## 🌐 Network Details
- Subnet 1: 192.168.40.0/25
- Subnet 2: 192.168.40.128/25

## ⚙️ Features
- Subnetting
- Router Configuration
- PC & Printer Connectivity

## 📊 Topology
![Network](topology.png)

## ▶️ How to Run
Open using Cisco Packet Tracer and test connectivity using ping.
